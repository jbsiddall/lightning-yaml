# API reference

Generated from the TSDoc in `src/`.

- **index** — the core parser: `parse`, `parseAll`, `stringify`.
- **js-yaml-compat** and **yaml-compat** — the drop-in shims. Their module pages
  carry the authoritative **compatibility matrix**: which options each library
  accepts, and how far our shim honours them today (most are accepted-but-ignored —
  see each page). Those module doc blocks are the master source for compatibility.
